Error List

This is not a page you need to check for errors. This page is just meant to keep track of the changes you make. Everytime you make an adjustment, be sure to note it below by modifying the tables below. I've already set up the tables to fit the correct number of changes that need to made. You just need to populate them with your changes.

Note: If you find any alt= that is not an error. It means that image is just decorative.
